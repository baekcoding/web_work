* 배울 프로그래밍 언어

1. 자바 스크립트 > 웹 브라우저의 동작을 프로그래밍 하기 위한 언어

2. 자바 > 웹 서버 앱을 프로그래밍 하기 위한 언어

3. 코틀린(가장 최근에 나옴) > 웹 서버 앱, 안드로이드 앱 모두 프로그래밍 가능

* 유틸리티 or 프레임워크 : 좀 더 개발을 체계적으로 하기 위한 틀

1. (자바) spring 프레임워크 : 웹 서버 앱을 체계적으로 만들기 위한 틀

2. (자스) vue js, react js : 웹 브라우저의 동작을 체계적으로 만들기 위한 틀

3. jquery(예전 꺼) : 웹 브라우저의 동작을 조금 더 쉽게 만들기 위한 유틸리티

콘솔(엔터 누르는 순간 바로 실행) : 자바 스크립트 실행하기 위한 공간

* 웹브라우저가 해석하는 문자열 3가지

1. html > 데이터 정보를 나열한다.

2. css > 디자인을 결정한다.

3. javascript > 언어적인 요소를 결정한다.
사용자와 상호 작용하는 UI를 만든다.
특정 시점에 어떠한 동작을 할 때 사용한다.

html은 head(웹 페이지의 설정 정보)와 body로 구성됨

block 요소(위에서 아래) : div, p, hn
inline 요소(좌에서 우로 일렬로) : img, button

순서 : html > data base > java > jsp > spring > react, vue
