서버는 이미 만들어져 있기 때문에 우리가 서버를 직접 만들 필요가 없다.
(메인 메소드도 우리가 만들 필요가 없다.)

다만 웹 클라이언트에서 요청할때 서버에서 어떤걸 가져오고 무엇을
웹에 전달할지만 우리가 프로그래밍을 통해 정하면 된다.
(자동차로 따지면 필요한 일부 부품만 제공하는 것)

클래스 : 객체를 설계하기 위한 설계도
=> 일부 부품의 설계도(클래스)만 만들어서 자동차 회사에 주면 자동차 회사에서
그 부품을 만들고 나머지 부품과 결합하여 자동차를 만들어 준다.

XML : 톰캣은 XML 문서를 읽어들여서 문서에 설정된 대로 준비한다.

톰캣이 사용하는 포트번호를 바꿔주는 이유 : 오라클에서 사용하는 포트번호(8080)와
같아서 충돌하기 때문 => 톰캣의 server.xml에서 변경

톰캣 서버 시작시켜서 웹페이지 확인
http://localhost:8888/Hello/index.html  => 웹앱 페이지
