* 리턴 없는 메소드(기능) 만드는법

public void drive() {
    System.out.println("달려요!")
}

==> javascript로 치면 function drive() {} 와 같다.

- public : 어디서도 접근이 가능하게 하겠다.

- void : 리턴 받을 값이 없다.


* 영역 구분

1. static
static 필드나 static 메소드가 클래스에 포장되어서 만들어지는 영역
클래스명을 이용해서 원하는 클래스를 찾아갈 수 있다.(클래스당 1개만 올라갈 수 있다.)

2. stack
메소드 안쪽이 실행될 때 메소드 안에서 만드는 지역 변수가 만들어지는 영역.
지역 변수명으로 원하는 값을 찾아서 사용할 수 있다.

3. heap
객체가 만들어져서 보관되는 사물함 영역.
각각의 객체는 사물함 키값(참조값)으로 구분된다.
즉, 참조값을 이용해서 원하는 객체를 찾아갈 수 있다.
